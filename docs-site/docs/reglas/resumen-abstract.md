# Resumen y Abstract

Referencia: Guía UNAP págs. 16-17

## Título "RESUMEN" / "ABSTRACT"

- 16 pt, centrado, negrita
- Interlineado 2.0
- Espaciado 0 / 10 pt

## Contenido

- 12 pt, justificado, normal (sin negrita)
- Interlineado 2.0
- **Extensión: 250 a 300 palabras** (estricto)
- Sin sangría

## "Palabras clave:" / "Keywords:"

- En **negrita**, primera letra mayúscula, dos puntos
- Las palabras: minúscula con primera letra mayúscula
- Separadas por comas (,)
- **Ordenadas alfabéticamente**

Ejemplo correcto:

```
Palabras clave: Identidad cultural, Patrimonio, Perspectivas, Tradiciones.
```

Ejemplo incorrecto:

```
Palabras Clave: tradiciones, identidad cultural, patrimonio   ❌ sin orden alfabético
```
