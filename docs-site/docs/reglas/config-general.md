# Configuración general del documento

## Referencia: Guía UNAP pág. 2

| Atributo | Valor obligatorio |
|----------|-------------------|
| Tamaño de papel | **A4** |
| Fuente | **Times New Roman** |
| Tamaño base | **12 pt** (excepto capítulos y títulos principales) |
| Color de texto | Negro |
| Color de página | Sin color |
| Margen izquierdo | **3.5 cm** (encuadernación) |
| Margen derecho | 2.5 cm |
| Margen superior | 2.5 cm |
| Margen inferior | 2.5 cm |
| Interlineado base | **2.0** |
| Sangría primera línea | **1.25 cm** por párrafo |
| Espaciado entre párrafos | Anterior 0 pt, posterior 10 pt |
| Viñetas permitidas | Letras (a.), guiones (-), puntos (•) |

## Numeración de páginas

!!! note "Regla específica"
    La numeración se cuenta mentalmente desde la portada hasta la lista de acrónimos, pero **se muestra visualmente solo a partir del RESUMEN**.

Páginas que NO deben mostrar número:

- Portada
- Reporte de Similitud
- Hoja de Jurados
- Dedicatoria
- Agradecimientos
- Índice General
- Índice de Tablas
- Índice de Figuras
- Índice de Anexos
- Lista de Acrónimos

Páginas que SÍ deben mostrar número:

- Resumen en adelante (incluyendo Anexos)

## Hojas en orientación horizontal (landscape)

Cuando una página rota a horizontal:

- El margen de encuadernación (3.5 cm) aparece visualmente **arriba**
- Los otros tres márgenes mantienen 2.5 cm
