# Tablas y Figuras

## Etiqueta "Tabla N" / "Figura N"

| Atributo | Valor |
|----------|-------|
| Tamaño | 12 pt |
| Estilo | **Negrita** |
| Alineación | **Izquierda** |
| Espaciado | 0 / 0 pt |
| Interlineado | 2.0 |
| Sangría izquierda | Según nivel del título contextual |

## Título descriptivo

| Atributo | Valor |
|----------|-------|
| Tamaño | 12 pt |
| Estilo | **Cursiva, SIN negrita** |
| Alineación | Izquierda |
| Espaciado | 0 / 0 pt |
| Interlineado | 2.0 |
| Sangría izquierda | Igual al nivel |

## La tabla/figura en sí

!!! warning "Alineación izquierda"
    La tabla/figura debe estar **alineada a la izquierda**, NO centrada. La sangría de su título es la que la posiciona horizontalmente según el nivel del título contextual.

### Tablas: encabezado y contenido

- **Primera fila (encabezado)**: Centrada y en negrita
- **Resto de filas**: NO en negrita (estilo normal)
- **Interlineado interno**: 1.0 o 1.5
- **Líneas horizontales**: presentes (formato APA)

### Tablas divididas en dos páginas

!!! danger "Encabezado repetido obligatorio"
    Si la tabla se extiende a dos páginas, la primera fila DEBE estar marcada como "Repetir como fila de encabezado en cada página".

    En Word: clic derecho sobre la primera fila → Propiedades de tabla → Fila → marcar "Repetir como fila de encabezado".

## Nota / Fuente

| Atributo | Valor |
|----------|-------|
| Tamaño | **10 pt** |
| Estilo | **Cursiva** (regla actualizada) |
| Puntuación | Dos puntos `:` obligatorios |
| Alineación | Izquierda |
| Espaciado anterior | 0 pt |
| Espaciado posterior | **15 pt** (crítico) |
| Interlineado | 1.5 |

Formato correcto: `Nota: descripción de la tabla.` o `Fuente: autor (año).`

## Secuencia completa

```
Tabla 1                       ← etiqueta (negrita, 12pt)
Título descriptivo            ← cursiva, 12pt, sin negrita
[ TABLA CON ENCABEZADO ]      ← primera fila negrita+centrada
Nota: información adicional.  ← cursiva, 10pt, posterior 15pt
```
