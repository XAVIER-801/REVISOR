# Preguntas frecuentes

## ¿El sistema reemplaza al revisor humano?

No. REVISOR verifica formato y estructura según la Guía UNAP 2.0. La revisión académica (originalidad, rigor metodológico, profundidad) sigue siendo responsabilidad de tu asesor y jurados.

## ¿Puedo usar el sistema antes de la versión final?

Sí, y se recomienda. Mientras más temprano detectes errores, menos trabajo tendrás al cierre.

## ¿Detecta plagio?

No directamente. Para verificar similitud usa **Turnitin**. Pero REVISOR sí detecta técnicas comunes de evasión:

- Texto en cuadros de texto
- Capturas de pantalla con texto

## ¿Mi archivo queda guardado?

No. El archivo se procesa en memoria efímera y se elimina inmediatamente. Solo se guardan estadísticas anónimas para mejorar el sistema.

## ¿Por qué algunas observaciones aparecen verdes?

Las verdes son sugerencias ortográficas (faltas tipográficas detectadas por el corrector). No son errores críticos pero corregirlas eleva la calidad.

## ¿Funciona con Maestría/Doctorado?

Sí. La Guía UNAP aplica a Pregrado, Maestría, Doctorado y Segundas Especialidades.

## ¿Por qué obtengo un score tan bajo?

El sistema es estricto. Una tesis bien formateada típicamente obtiene 85+. Lo importante no es el número, sino que no queden errores críticos (rojos).

## ¿Cómo se calcula el puntaje?

`score = (validaciones aprobadas / total de reglas) × 100`

Cada categoría tiene múltiples reglas. Mientras más reglas pasen, mayor el score.

## ¿Funciona si mi tesis tiene fórmulas matemáticas?

Sí. El motor detecta fórmulas OMML y las trata como elementos especiales (no las audita como texto).
